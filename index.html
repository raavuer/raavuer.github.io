Role and Task Generator

Role and Task Generator (RTG) was made for the general purpose of streamlining the process of pulling names from a hat.

In online games, computers can automate much of the work done, but when playing games in real life, we have to do it ourselves, which can be tedious.

Join Room

Once a room has been created, you can join it by clicking the "Join Room" button above.

To join a room, you must enter a username and the room code.

Host Room

To start using RTG, first, you need to host a room. Only one person can be the host of a room, the other players join it.

To host a room, click the "Host Room" button above. Then, the host will be able to set the room settings, and share the room code for others to join.

The host can choose to play or not. If the host does, they will need to provide a username.

In Game

When in a room, the users wait for the host to start the game.

Once started, everyone will be given a role and tasks, predefined by the host.

Because this is meant to be used for real life games, players will be able to hide their role for those looking over peoples shoulders.
