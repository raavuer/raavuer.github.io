Host Room

To start using RTG, first, you need to host a room. Only one person can be the host of a room, the other players join it.

To host a room, click the "Host Room" button above. Then, the host will be able to set the room settings, and share the room code for others to join.

The host can choose to play or not. If the host does, they will need to provide a username.
